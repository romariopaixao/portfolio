Tecnologias que estou aprendendo

Atualmente, estou focando nos seguintes tópicos e ferramentas para fortalecer minha carreira em desenvolvimento de software:

  • ✅ Flutter — desenvolvimento mobile com uma única base de código para Android e iOS.
  • ✅ Fullstack — Desenvolvedora capaz de atuar no frontend e backend de aplicações, construindo desde a interface até a lógica de funcionamento.
  • ✅ JavaScript — criação de interatividade no frontend e base para frameworks como React.
  • ✅ SQL — manipulação de dados e bancos relacionais.
  • ✅ Versionamento com Git — essencial para projetos em equipe e controle de código.

Dicas para quem está começando

  • ➡️ Não tente aprender tudo de uma vez. Escolha uma stack e vá fundo!
  • ➡️ Pratique mais do que consome conteúdo. A prática fixa o conhecimento.
  • ➡️ Use o GitHub como seu portfólio vivo. Commits frequentes mostram evolução.
  • ➡️ Não tenha medo de errar. Aprendizado vem dos erros também.
  • ➡️ Participe de comunidades. Você não precisa aprender sozinha(o)!
